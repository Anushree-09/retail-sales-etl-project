# Retail Sales ETL & Power BI Dashboard

This project demonstrates an end-to-end data pipeline and analytics dashboard.

## Project Architecture
CSV Dataset → Python ETL → MySQL Database → Power BI Dashboard

## Tools Used
- Python (Pandas)
- MySQL
- Power BI
- VS Code

## Features
- Data extraction and transformation using Python
- Data storage in MySQL database
- Interactive Power BI dashboard analyzing:
  - Sales trends
  - Regional profit
  - Top products
  - Sales distribution by state

## Dashboard
Open the .pbix file in Power BI Desktop to explore the dashboard.
